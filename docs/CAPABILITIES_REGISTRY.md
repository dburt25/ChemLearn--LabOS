# Capabilities Registry

Master list of scientific and operational capabilities that LabOS offers.

## Structure
- `capability_id`
- `description`
- `owner_bot_role`
- `status` (planned/in-progress/available/deprecated)
- `validation_reference`

## Initial Entries (to be populated)
- PChem foundational calculators
- EI-MS spectral interpreter
- Proteomics digest planner
- Simulation engine hooks

## Maintenance Rules
- Registry updated whenever a capability is added, modified, or retired.
- Validation evidence linked to `VALIDATION_LOG.md` (Phase 1+).
