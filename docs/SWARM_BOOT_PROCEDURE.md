# Swarm Boot Procedure

Checklist for spinning up a new ChemLearn LabOS swarm session.

## Pre-Boot
1. Sync repository (pull main + review docs).
2. Confirm Git LFS installed and configured.
3. Review `PHASES_OVERVIEW.md` to ensure correct context.

## Boot Steps
1. Assign roles and reference `SWARM_PERMISSIONS_MATRIX.md`.
2. Log session start (timestamp, participants, goals) in `compliance-notes.md`.
3. Outline intended changes and approval path.

## Shutdown Steps
1. Summarize work performed and remaining tasks.
2. Update applicable docs (CHANGELOG, VALIDATION_LOG once available).
3. Create or link to audit entries for reproducibility.
